# GardenMonitoring
Web application that get and show information about certain plants and their condition with realised authentication, authorization and system of roles (_agronomist_ and _agricultural worker_)
## Used techonologies
* _ASP.NET MVC_
* _Entity Framework_
* _MS SQL_
